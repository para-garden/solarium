# solarium

Self-contained works across any medium — each built around a single premise, complete in itself.

## What this is

Each piece is an island. No shared universe, no ongoing world, no accumulated narrative debt. The premise is the whole justification.

Medium varies per piece: text, interactive, visual, audio — whatever serves it.
